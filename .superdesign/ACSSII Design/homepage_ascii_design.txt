## Visual Hierarchy & UX Patterns

1. Information Architecture: Device control → Sleep data → Navigation
2. Primary Actions: Device controls are prominently displayed in gradient cards
3. Secondary Actions: Navigation tabs provide access to other app sections
4. Data Visualization: Metrics use color-coded icons and trend indicators
5. Interactive Elements: Cards have hover states and subtle shadows
6. Responsive Design: Flexbox layout adapts to content

## State Management
- Connected Device: Green status dot with "Connected" text
- Control Values: Real-time display (Volume: 0, Fan: 9, Light: 0)
- Sleep Quality: Numerical score with qualitative badge
- Metric Trends: Visual indicators for positive/negative changes

## Key Design Decisions
1. Phone bezel design creates realistic mobile app preview
2. Gradient backgrounds add visual depth and modern aesthetic
3. Card-based layout organizes information clearly
4. Icon usage provides quick visual recognition
5. Color coding enhances data comprehension
6. Bottom navigation enables easy access to main features
